## 🎨 디자인 하이라이트

### 컬러 시스템
```css
Primary: #a2c54e (연두색)
├─ 50:  #f4f9e8
├─ 100: #e6f3c8
├─ 500: #a2c54e ← Main
├─ 600: #8fb045
└─ 900: #506828

Semantic:
├─ Success: #22c55e
├─ Warning: #f59e0b
├─ Error: #ef4444
└─ Info: #3b82f6
```

### 타이포그래피
- **폰트**: Pretendard (한글 최적화)
- **크기 계층**:
  - Headline: 3xl (30px)
  - Title: xl (20px)
  - Body: base (16px)
  - Caption: sm (14px)

### Border Radius
- **2xl**: 24px (주요 카드, 버튼)
- **3xl**: 32px (강조 요소)
- **xl**: 16px (작은 요소)

### Shadow System
- **soft**: 미묘한 그림자 (일반 카드)
- **medium**: 중간 그림자 (강조 카드)
- **lift**: 호버 시 들어올림 효과

## 🎬 애니메이션 & 인터랙션

### Framer Motion 활용
1. **Page Transition**: Fade + Slide up (10px, 0.3s)
2. **Stagger Animation**: 대시보드 요소들이 순차적으로 등장
3. **Button Tap**: Scale 0.98
4. **Hover Lift**: TranslateY(-4px) + Shadow

### Micro-interactions
- **Input Focus**: Border color 변화 + Ring 효과
- **Button Hover**: Lift + Shadow 강화
- **Card Hover**: Lift + Border 변화
- **Loading Spinner**: 회전 애니메이션
