# 마이페이지 고도화 및 로고 복구 - 10

## 작업 개요

App.tsx의 로고 경로를 `/images/shingu_logo.png`로 수정하고, MyPage.tsx에 도서관 대출 현황 섹션을 추가하여 현재 대출 중인 도서 목록과 반납 예정일을 확인할 수 있도록 개선했습니다.

---

## 주요 변경 사항

### 1. App.tsx 로고 경로 수정

#### 최종 경로
```tsx
<img 
  src="/images/shingu_logo.png"  // /shingu_logo.png에서 변경
  alt="신구대학교 로고" 
  className="h-10"
  onError={(e) => {
    e.currentTarget.style.display = 'none';
    const fallback = e.currentTarget.nextElementSibling as HTMLElement;
    if (fallback) fallback.style.display = 'block';
  }}
/>
<div className="hidden text-lg font-bold text-gray-900 dark:text-white">
  신구대학교
</div>
```

**변경 사항:**
- 경로: `/shingu_logo.png` → `/images/shingu_logo.png`
- 이유: public 폴더 내 images 디렉토리 구조에 맞춤

**Fallback 동작:**
1. 이미지 로드 성공 → 로고 표시
2. 이미지 로드 실패 → "신구대학교" 텍스트 표시
3. onError 핸들러가 자동으로 이미지 숨기고 텍스트 표시

### 2. 도서관 대출 현황 섹션 추가

#### 위치
- 좌측 컬럼
- 학적 정보 섹션과 기본 정보 섹션 사이
- transition delay: 0.25

#### 구조
```tsx
<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ delay: 0.25 }}
  className="bg-white dark:bg-gray-800 rounded-3xl shadow-sm p-8"
>
  <div className="flex items-center gap-3 mb-6">
    <Library className="w-6 h-6 text-primary-600" />
    <h3 className="text-2xl font-bold">도서관 대출 현황</h3>
  </div>

  {/* 대출 도서 리스트 */}
  <div className="space-y-4">
    {books.map((book, index) => (
      <div className="p-5 bg-gray-50 rounded-2xl hover:bg-primary-50 transition">
        <div className="flex items-start justify-between gap-4">
          {/* 도서 정보 */}
          <div className="flex-grow">
            <h4>{book.title}</h4>
            <p>저자: {book.author}</p>
            <p>반납 예정: {book.dueDate}</p>
          </div>

          {/* 배지 */}
          <div>
            {book.daysLeft <= 3 ? (
              <div className="bg-error/10 text-error rounded-full">
                <AlertCircle />
                <span>반납 {book.daysLeft}일 전</span>
              </div>
            ) : (
              <div className="bg-gray-200 text-gray-700 rounded-full">
                <span>{book.daysLeft}일 남음</span>
              </div>
            )}
          </div>
        </div>
      </div>
    ))}
  </div>

  {/* 대출 요약 */}
  <div className="mt-6 p-4 bg-primary-50 rounded-2xl">
    <span>총 대출 도서</span>
    <span className="text-xl font-bold text-primary-600">3권</span>
  </div>
</motion.div>
```

#### 대출 도서 데이터
```tsx
const books = [
  { 
    title: '클린 코드', 
    author: '로버트 C. 마틴', 
    dueDate: '2026-01-16', 
    daysLeft: 3  // ⚠️ 경고 배지
  },
  { 
    title: '리팩토링', 
    author: '마틴 파울러', 
    dueDate: '2026-01-18', 
    daysLeft: 5  // ✅ 일반 배지
  },
  { 
    title: '이펙티브 자바', 
    author: '조슈아 블로크', 
    dueDate: '2026-01-20', 
    daysLeft: 7  // ✅ 일반 배지
  }
];
```

#### 배지 로직
```tsx
{book.daysLeft <= 3 ? (
  // ⚠️ 경고 배지 (3일 이하)
  <div className="flex items-center gap-1 px-3 py-1 bg-error/10 text-error rounded-full">
    <AlertCircle className="w-4 h-4" />
    <span className="text-xs font-semibold">반납 {book.daysLeft}일 전</span>
  </div>
) : (
  // ✅ 일반 배지 (4일 이상)
  <div className="px-3 py-1 bg-gray-200 dark:bg-gray-600 text-gray-700 dark:text-gray-300 rounded-full">
    <span className="text-xs font-semibold">{book.daysLeft}일 남음</span>
  </div>
)}
```

**배지 종류:**
1. **경고 배지** (3일 이하):
   - 배경: `bg-error/10` (빨간색 10%)
   - 텍스트: `text-error` (빨간색)
   - 아이콘: AlertCircle
   - 메시지: "반납 X일 전"

2. **일반 배지** (4일 이상):
   - 배경: `bg-gray-200`
   - 텍스트: `text-gray-700`
   - 아이콘: 없음
   - 메시지: "X일 남음"

## 사용자 경험 (UX)

### 시각적 주의 환기
**경고 배지 (3일 이하):**
- 빨간색 배경 + 빨간색 텍스트
- AlertCircle 아이콘
- "반납 X일 전" 메시지
→ 즉각적인 주의 환기

**일반 배지 (4일 이상):**
- 회색 배경 + 회색 텍스트
- "X일 남음" 메시지
→ 여유 있는 상태 표시

### 정보 계층
```
도서관 대출 현황
├── 클린 코드 ⚠️ 반납 3일 전
├── 리팩토링 (5일 남음)
├── 이펙티브 자바 (7일 남음)
└── 총 대출 도서: 3권
```

**우선순위:**
1. 경고 배지 (가장 눈에 띔)
2. 도서 제목
3. 저자 및 반납일
4. 총 대출 수

### 인터랙션
- Hover 시 연두색 배경 (`hover:bg-primary-50`)
- 부드러운 transition
- 클릭 가능한 느낌 (향후 상세 페이지 연결 가능)
