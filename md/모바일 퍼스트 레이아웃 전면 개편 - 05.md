# 모바일 퍼스트 레이아웃 전면 개편 - 05

## 작업 개요

App.tsx와 Home.tsx를 **모바일 퍼스트(Mobile-First)** 관점에서 전면 개편했습니다. 상단 유틸리티 바 추가, 사이드바 메뉴 최적화, 디지털 ID 카드 재설계, 대시보드 간소화를 통해 **Simple is Best** 원칙을 실현했습니다.

---

## 주요 개선 사항

### 1. 상단 유틸리티 바 (Utility Bar)

#### 추가된 퀵 링크
- **Office 365** (파란색, FileText 아이콘)
- **Gmail** (빨간색, Mail 아이콘)  
- **사이버캠퍼스** (초록색, Globe 아이콘)
- **인터넷 증명발급** (보라색, FileText 아이콘)

#### 반응형 디자인
**데스크톱 (≥md):**
```typescript
<div className="hidden md:flex items-center gap-1">
  <a className="flex items-center gap-2">
    <div className={link.color}>{link.icon}</div>
    <span className="text-xs hidden lg:block">{link.name}</span>
  </a>
</div>
```
- 아이콘 + 텍스트 결합
- lg 이상에서 텍스트 표시

**모바일 (<md):**
```typescript
<div className="flex md:hidden items-center gap-1">
  <a className="p-2">
    <div className={link.color}>{link.icon}</div>
  </a>
</div>
```
- 아이콘만 표시
- 공간 절약

### 4. 디지털 ID 카드 재설계

#### Mobile-First 레이아웃
```typescript
<div className="p-8">
  {/* 헤더 */}
  <div className="text-center mb-6">
    <span>신구대학교 디지털 학생증</span>
  </div>

  {/* QR 코드 - 상단 중앙 */}
  <div className="flex justify-center mb-8">
    <div className="bg-white p-5 rounded-2xl">
      <QRCodeSVG size={160} />
    </div>
  </div>

  {/* 학생 정보 - 중앙 정렬 */}
  <div className="text-center space-y-4">
    <h2 className="text-4xl font-bold">{name}</h2>
    <div><Hash /> {studentId}</div>
    <p className="text-2xl font-bold">{department}</p>
    <div><Award /> {year}</div>
  </div>
</div>
```

**Before vs After:**
```
Before (3단 그리드):
┌──────────┬──────┐
│ 학생정보 │ QR   │
│ 2/3     │ 1/3  │
└──────────┴──────┘

After (중앙 정렬):
┌────────────────┐
│      QR        │
│                │
│    김민수      │
│    #123        │
│ IT소프트웨어과 │
│    3학년       │
└────────────────┘
```

**장점:**
- 모바일에서 가독성 향상
- QR 코드가 가장 먼저 보임
- 깔끔한 타이포그래피

### 5. Home 대시보드 간소화

#### ❌ 삭제된 요소
- 하단 퀵 링크 배너 (AI 챗봇, 학점 계산기, 강의 리뷰)
- Quick Stats 카드 3종 (학점 이수율, GPA, 수강 학점)

#### ✅ 유지된 핵심 요소

**1) 디지털 ID 카드** (최상단)
- QR 코드 중앙 배치
- 학생 정보 한눈에

**2) 오늘의 수업** (좌측)
- 타임라인 UI 유지
- **현재 수업 강조 강화:**
  ```typescript
  {schedule.isNow 
    ? 'w-5 h-5 bg-primary-500 ring-4 ring-primary-200 animate-pulse'
    : 'w-4 h-4 bg-gray-300'}
  
  {schedule.isNow
    ? 'bg-gradient-to-br from-primary-50 to-primary-100 border-2 border-primary-500 shadow-md'
    : 'bg-gray-50 border-2 border-transparent'}
  ```

**특징:**
- 타임라인 점 크기: 4px → 5px (현재 수업)
- Ring 효과 + Pulse 애니메이션
- 그라데이션 배경
- 테두리 2px
- 그림자 추가

**3) 최신 공지사항** (우측)
- **탭 필터 추가:**
  ```typescript
  const [noticeTab, setNoticeTab] = useState<'all' | 'academic' | 'scholarship'>('all');
  
  <div className="flex gap-2 border-b">
    <button onClick={() => setNoticeTab('all')}>전체</button>
    <button onClick={() => setNoticeTab('academic')}>학사</button>
    <button onClick={() => setNoticeTab('scholarship')}>장학</button>
  </div>
  
  const filteredNotices = noticeTab === 'all' 
    ? allNotices 
    : allNotices.filter(n => n.category === noticeTab);
  ```

## 디자인 원칙: Simple is Best

### 1. 여백(Padding) 충분히 활용
```typescript
// Home.tsx
<div className="space-y-8"> // 8 * 0.25rem = 2rem = 32px

// Cards
<div className="p-8"> // 2rem = 32px padding

// Schedule items
<div className="space-y-6"> // 1.5rem = 24px gap
```

**Before vs After:**
```
Before: space-y-6 (24px)
After:  space-y-8 (32px)

Card padding: 6 → 8 (24px → 32px)
```

### 2. Rounded-3xl 일관 적용
```typescript
// 모든 카드
className="rounded-3xl" // 32px border radius

// 버튼, 입력 필드
className="rounded-2xl" // 24px border radius
```

### 3. 은은한 그림자
```typescript
// Cards
className="shadow-sm" // 매우 은은한 그림자

// Not using:
// shadow-md, shadow-lg, shadow-xl
```
