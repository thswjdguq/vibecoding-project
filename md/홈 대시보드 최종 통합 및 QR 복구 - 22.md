## 주요 개선 사항

### 1. 용어 및 텍스트 개선

#### Before → After
- ❌ '긴급 공지' → ✅ **'주요 학사공지'**
- ❌ '성적 현황' → ✅ **'나의 학업 리포트'**

#### 공지 배지 변경
**Before: NEW 배지**
```tsx
<span className="px-2 py-1 bg-error text-white text-xs font-bold rounded-full">
  NEW
</span>
```

**After: 카테고리 배지**
```tsx
<span className={`px-2.5 py-1 text-xs font-bold rounded-full ${
  notice.category === '학사' 
    ? 'bg-blue-100 text-blue-700'
    : 'bg-primary-100 text-primary-700'
}`}>
  {notice.category}  {/* 학사, 장학 */}
</span>
```

**장점:**
- 정보 우선순위 명확
- 카테고리별 색상 구분
- 더 부드러운 느낌

### 2. QR 위젯 복구 및 배치

#### SimpleQRWidget import
```tsx
import SimpleQRWidget from '../components/SimpleQRWidget';

<SimpleQRWidget 
  name={userName}
  studentId={studentId}
  department={department}
  year={year}
/>
```

#### 데스크톱: Sticky 우측 사이드바
```tsx
<div className="lg:flex lg:gap-6">
  {/* 메인 콘텐츠 */}
  <div className="flex-1">
    {/* 위젯들 */}
  </div>

  {/* QR 위젯 (데스크톱 우측) */}
  <div className="hidden lg:block lg:w-80 lg:sticky lg:top-24 lg:self-start">
    <SimpleQRWidget {...props} />
  </div>
</div>
```

**특징:**
- `lg:sticky`: 스크롤 시 고정
- `lg:top-24`: 상단 여백
- `lg:w-80`: 320px 고정 너비
- `lg:self-start`: 상단 정렬

#### 모바일: 최상단 배치
```tsx
{/* QR 위젯 (모바일 상단) */}
<div className="lg:hidden mb-6">
  <SimpleQRWidget {...props} />
</div>
```

**특징:**
- AI 그리팅 카드 바로 위
- 앱 실행 즉시 인증 가능
- 모바일 우선 설계

### 3. 학사일정 위젯 (NEW!)

```tsx
const academicSchedule = [
  { event: '중간고사 기간', date: '2026-04-15', daysLeft: 92, type: '시험' },
  { event: '종강', date: '2026-06-20', daysLeft: 158, type: '학사' },
  { event: '여름학기 수강신청', date: '2026-05-10', daysLeft: 117, type: '수강' }
];

<div className="bg-white rounded-3xl p-6 shadow-sm">
  <div className="flex items-center justify-between mb-4">
    <h3 className="font-bold text-lg">다가오는 학사일정</h3>
    <Calendar className="w-5 h-5 text-purple-600" />
  </div>
  <div className="space-y-3">
    {academicSchedule.map((schedule) => (
      <div className="flex items-center justify-between p-4 bg-gray-50 rounded-2xl">
        <div>
          <p className="font-semibold">{schedule.event}</p>
          <div className="flex items-center gap-2 text-sm text-gray-500">
            <Calendar className="w-3.5 h-3.5" />
            <span>{schedule.date}</span>
          </div>
        </div>
        <div className="text-right">
          <span className={`inline-block px-3 py-1 rounded-full text-xs font-bold ${
            schedule.type === '시험'
              ? 'bg-error text-white'
              : schedule.type === '수강'
              ? 'bg-blue-500 text-white'
              : 'bg-purple-500 text-white'
          }`}>
            {schedule.type}
          </span>
          <p className="text-xs text-gray-500 mt-1">D-{schedule.daysLeft}</p>
        </div>
      </div>
    ))}
  </div>
</div>
```

**정보 구조:**
- 일정명: 중간고사, 종강, 수강신청
- 날짜: YYYY-MM-DD
- D-Day: 남은 일수
- 타입 배지: 시험(빨강), 수강(파랑), 학사(보라)

### 4. 레이아웃 밸런싱

#### Before: 3열 그리드만
```
[학식] [공지] [도서관]
[======과제======] [성적]
```

#### After: 3열 + 2열 그리드 조합
```
[학식] [공지] [도서관]
[====과제====] [====학사일정====]
[========학업 리포트========]
```

**코드:**
```tsx
{/* 메인 위젯 3열 그리드 */}
<div className="grid grid-cols-1 md:grid-cols-3 gap-6">
  {/* 학식, 공지, 도서관 */}
</div>

{/* 2열 그리드: 과제 + 학사일정 */}
<div className="grid grid-cols-1 md:grid-cols-2 gap-6 mt-6">
  {/* 마감 임박 과제 */}
  {/* 다가오는 학사일정 */}
</div>

{/* 학업 리포트 (full width) */}
<div className="mt-6 bg-white rounded-3xl ...">
  {/* 성적 통계 */}
</div>
```

**장점:**
- 화면 공간 효율적 활용
- 중요도별 그룹핑
- 과제와 일정 나란히 비교 가능

### 5. 나의 학업 리포트 (개선)

#### Before: 세로 배치
```
3.85
전체 평점

전공: 4.0
학점: 78/140
```

#### After: 가로 3분할
```tsx
<div className="grid grid-cols-3 gap-6">
  <div className="text-center">
    <p className="text-5xl font-bold text-primary-600 mb-2">3.85</p>
    <p className="text-sm text-gray-600">전체 평점</p>
  </div>
  <div className="text-center border-l border-r border-gray-200">
    <p className="text-5xl font-bold text-blue-600 mb-2">4.0</p>
    <p className="text-sm text-gray-600">전공 평점</p>
  </div>
  <div className="text-center">
    <p className="text-3xl font-bold text-purple-600 mb-2">78 / 140</p>
    <p className="text-sm text-gray-600">취득 학점</p>
  </div>
</div>
```

**특징:**
- 세로 구분선: `border-l border-r`
- 색상 구분: 연두(전체), 파랑(전공), 보라(학점)
- 큰 숫자: `text-5xl`
- 시각적 밸런스

## 반응형 레이아웃

### 모바일 (1열)
```
┌────────────┐
│ QR 위젯    │ ← 최상단
├────────────┤
│ AI 그리팅  │
├────────────┤
│ 퀵 액션    │
├────────────┤
│ 학식       │
├────────────┤
│ 공지       │
├────────────┤
│ 도서관     │
├────────────┤
│ 과제       │
├────────────┤
│ 학사일정   │
├────────────┤
│ 학업리포트 │
└────────────┘
```

### 데스크톱 (3열 + 사이드바)
```
┌─────────────────────────────┬───────┐
│ AI 그리팅                   │  QR   │
├─────────────────────────────┤       │
│ 퀵 액션                     │ 위젯  │
├─────────┬─────────┬─────────┤       │
│ 학식    │ 공지    │ 도서관  │(고정) │
├─────────┴─────────┴─────────┤       │
│ 과제      │ 학사일정         │       │
├───────────────────────────  ┤       │
│ 학업 리포트                 │       │
└─────────────────────────────┴───────┘
```

**Sticky 동작:**
- QR 위젯: 스크롤해도 우측 고정
- `top-24`: 헤더 아래 고정
- `self-start`: 상단 정렬
