### 2. 강의 개요 카드

#### 강의 목표
```tsx
<div className="bg-white dark:bg-gray-800 rounded-3xl shadow-sm p-8">
  <div className="flex items-center gap-3 mb-6">
    <div className="w-12 h-12 rounded-full bg-primary-100 flex items-center justify-center">
      <BookOpen className="w-6 h-6 text-primary-600" />
    </div>
    <h2 className="text-2xl font-bold">강의 개요</h2>
  </div>
  
  {/* 강의 목표 */}
  <div className="mb-6">
    <div className="flex items-center gap-2 mb-3">
      <Target className="w-5 h-5 text-primary-600" />
      <h3 className="font-semibold text-lg">강의 목표</h3>
    </div>
    <p className="text-gray-700 leading-relaxed">
      본 강의는 웹 프로그래밍의 기초부터 실무 활용까지 체계적으로 학습합니다. 
      HTML, CSS, JavaScript의 기본 개념을 이해하고, React 프레임워크를 활용한 
      현대적인 웹 애플리케이션 개발 능력을 배양합니다.
    </p>
  </div>
</div>
```

**특징:**
- BookOpen 아이콘: 12x12 원형, 연두색 배경
- Target 아이콘: 목표 강조
- 넉넉한 줄 간격: `leading-relaxed`

#### 사용 교재
```tsx
<div>
  <h3 className="font-semibold text-lg mb-3">사용 교재</h3>
  <div className="bg-gray-50 dark:bg-gray-700 rounded-2xl p-4">

    </p>
  </div>
</div>
```

**디자인:**
- 회색 배경 카드
- 책 이모지
- 작은 폰트로 상세 정보

### 3. 평가 비중 위젯 (Visual Bars)

```tsx
<div className="bg-white dark:bg-gray-800 rounded-3xl shadow-sm p-8">
  <h2 className="text-2xl font-bold mb-6">평가 비중</h2>
  <div className="space-y-4">
    {[
      { name: '출석', weight: 10, color: 'bg-blue-500' },
      { name: '중간고사', weight: 30, color: 'bg-purple-500' },
      { name: '기말고사', weight: 40, color: 'bg-primary-500' },
      { name: '과제', weight: 20, color: 'bg-orange-500' }
    ].map((item) => (
      <div key={item.name}>
        <div className="flex items-center justify-between mb-2">
          <span className="font-semibold">{item.name}</span>
          <span className="text-lg font-bold text-primary-600">
            {item.weight}%
          </span>
        </div>
        <div className="w-full h-3 bg-gray-200 rounded-full overflow-hidden">
          <div
            className={`h-full ${item.color} transition-all duration-500`}
            style={{ width: `${item.weight}%` }}
          />
        </div>
      </div>
    ))}
  </div>
</div>
```

**시각적 바 특징:**
- 높이: `h-3` (12px)
- 배경: `bg-gray-200` (회색)
- 전환: `transition-all duration-500`
- 동적 너비: `style={{ width: `${item.weight}%` }}`

**색상 구분:**
- 출석: 파랑 (`bg-blue-500`)
- 중간고사: 보라 (`bg-purple-500`)
- 기말고사: 연두 (`bg-primary-500`) ← 가장 중요
- 과제: 주황 (`bg-orange-500`)

**비율:**
```
출석:      10% ████████░░
중간고사:  30% ████████████████████████░░░░░░
기말고사:  40% ████████████████████████████████░░░░░░░░░░
과제:      20% ████████████████░░░░░░░░░░░░░░
```

### 4. 주차별 수업 계획 (15주)

```tsx
<div className="bg-white dark:bg-gray-800 rounded-3xl shadow-sm p-8">
  <h2 className="text-2xl font-bold mb-6">주차별 수업 계획</h2>
  <div className="overflow-x-auto">
    <table className="w-full">
      <thead>
        <tr className="border-b-2 border-gray-200">
          <th className="text-left py-4 px-4 font-semibold w-24">주차</th>
          <th className="text-left py-4 px-4 font-semibold">학습 주제</th>
          <th className="text-left py-4 px-4 font-semibold w-32">수업 방식</th>
        </tr>
      </thead>
      <tbody>
        {[
          { week: 1, topic: '웹 프로그래밍 개요 및 HTML 기초', method: '이론' },
          { week: 2, topic: 'CSS 스타일링과 레이아웃', method: '이론+실습' },
          // ... 15주까지
        ].map((item) => (
          <tr key={item.week} className="border-b hover:bg-gray-50 transition">
            <td className="py-4 px-4">
              <span className="inline-flex items-center justify-center 
                w-10 h-10 rounded-full bg-primary-100 
                text-primary-700 font-bold">
                {item.week}
              </span>
            </td>
            <td className="py-4 px-4">{item.topic}</td>
            <td className="py-4 px-4">
              <span className={`px-3 py-1 rounded-full text-sm font-semibold ${
                item.method === '시험' 
                  ? 'bg-error text-white'
                  : item.method === '실습'
                  ? 'bg-primary-100 text-primary-700'
                  : 'bg-gray-200 text-gray-700'
              }`}>
                {item.method}
              </span>
            </td>
          </tr>
        ))}
      </tbody>
    </table>
  </div>
</div>
```

**주차 번호 디자인:**
- 원형 배지: `w-10 h-10 rounded-full`
- 배경: `bg-primary-100` (연한 연두)
- 텍스트: `text-primary-700` (진한 연두)
- 굵은 폰트: `font-bold`

**수업 방식 배지:**
- **시험** (8주, 15주): `bg-error text-white` (빨간 배경)
- **실습**: `bg-primary-100 text-primary-700` (연두)
- **이론** / **이론+실습**: `bg-gray-200` (회색)

**15주 커리큘럼:**
1. 웹 프로그래밍 개요 및 HTML 기초
2. CSS 스타일링과 레이아웃
3. JavaScript 기초 문법
4. DOM 조작과 이벤트 처리
5. jQuery와 Ajax
6. 프론트엔드 프레임워크 개요
7. React 기초
8. **중간고사** ← 빨간 배지
9. React 컴포넌트와 Props
10. State와 라이프사이클
11. React Hooks
12. React Router와 네비게이션
13. API 연동과 비동기 처리
14. 프로젝트 실습
15. **기말고사** ← 빨간 배지

## 사용자 경험

### 수업 계획표 보기 Flow
```
1. E-Class 메뉴 클릭
   ↓
2. 원하는 과목 선택
   ↓
3. 자동으로 "수업 계획표" 탭 표시
   ↓
4. 강의 목표 확인
   ↓
5. 교재 정보 확인
   ↓
6. 평가 비중 시각적 확인 (바 차트)
   ↓
7. 주차별 계획 스크롤
   ↓
8. 필요 시 다른 탭으로 이동
```

### 학생 입장 장점
1. **한눈에 파악**: 첫 화면에 계획표
2. **시각적 정보**: 평가 비중 바 차트
3. **체계적**: 15주 전체 계획
4. **강조**: 시험 주차 빨간 배지
